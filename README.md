# SBA 308: JavaScript Fundamentals

## What is this?

This project helps process and calculate learners' scores for assignments. It checks if the data is valid, filters out assignments that aren’t due yet, and handles late submissions by applying a 10% penalty. Then it calculates the weighted average score for each learner.

## Features

- Validates course and assignment data.
- Filters assignments that aren’t due yet.
- Deducts 10% for late submissions.
- Calculates weighted average scores for learners.
